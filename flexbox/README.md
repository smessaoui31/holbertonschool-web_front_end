# Flexbox

## Description

This project is a series of exercises to learn and practice CSS Flexbox. Starting from a static HTML and CSS base, each task progressively introduces flexbox properties to replace older layout techniques like floats and inline-block.

## Learning Objectives

- Understand what Flexbox is and how it works
- Convert float-based layouts to flexbox
- Use flex container properties: display, flex-direction, flex-wrap, justify-content, align-items
- Use flex item properties: align-self
- Know the difference between the main axis and the cross axis

## Tasks

| Task | Files | Description |
|------|-------|-------------|
| 0 | 0-index.html, 0-styles.css | Add display flex on the row class |
| 1 | 1-index.html, 1-styles.css | Add new classes on each section |
| 2 | 2-index.html, 2-styles.css | Reverse order of latest news cards |
| 3 | 3-index.html, 3-styles.css | Simplify services list with flex-wrap |
| 4 | 4-index.html, 4-styles.css | Adjust spacing between flex items |
| 5 | 5-index.html, 5-styles.css | Flexify the header |
| 6 | 6-index.html, 6-styles.css | Flexify the navbar |
| 7 | 7-index.html, 7-styles.css | Align center logo and navbar |
| 8 | 8-index.html, 8-styles.css | Simplify the hero banner with flexbox |
| 9 | 9-index.html, 9-styles.css | Center about us columns with align-self |
